# price-tracker-power-automate
End-to-end price tracking system using Power Automate, Excel, and Power BI with automated scraping and trend analysis
## 🔧 Features
- Automated price extraction from Amazon
- Intelligent product matching (handles name variations)
- Price comparison with historical tracking
- Automated percentage change calculation
- Interactive Power BI dashboard for insights

## 🛠 Tech Stack
- Power Automate Desktop
- Excel
- Power BI
